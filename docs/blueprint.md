# **App Name**: BarberFlow

## Core Features:

- Barbershop Admin Registration: Allows barbershop administrators to register and manage their barbershop's profile, including name, address, and contact information.
- Barber/Staff Registration: Enables barbershop administrators to register barbers with their names, work schedules.
- Client Registration: Allows clients to register with their name, WhatsApp phone number (mandatory), and optional email.
- Service and barber Appointment Booking: Enables clients to book appointments by selecting barbershop, service, barber (or 'Anyone'), date, and available time slots, and save booking details on Firestore
- Appointment Confirmation via WhatsApp: Uses a third-party API to send immediate appointment confirmation messages via WhatsApp to the client after booking. This feature requires using the WhatsApp phone number of clients that are captured upon registration.
- Appointment Reminder via WhatsApp: Uses a third-party API to send appointment reminder messages via WhatsApp 2 hours before the scheduled time.
- Smart Shift: Predict availability for appointment schedules using AI

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) for trust and reliability.
- Background color: Light gray (#F0F2F5) for a clean and modern look.
- Accent color: A teal (#009688) to provide a clear visual contrast, emphasizing important features like call to action buttons.
- Body and headline font: 'Inter', a grotesque-style sans-serif.
- Use modern, outline-style icons related to barbershop services, such as scissors, razors, and chairs.
- Clean and modern layout with clear separation of sections for easy navigation.
- Subtle transitions and animations on button hover and page transitions to enhance user experience.
